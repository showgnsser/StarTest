# GPS L5 - 状态门限徘徊

固定案例 ID：`ST-GPSL5-07-THRESHOLD_HYSTERESIS`

## 现实场景

验证C/N0在状态门限附近波动时的迟滞和抗抖动能力。

## 输入

- 信号：`GPS L5`
- 数据源：StarGen 实时二进制管道
- 单次时长：`900 s`
- 固定种子：`20260716, 20260717, 20260718`
- 输入过程：在每个强弱切换门限上下交替变化并保持足够长时间。

## 真值

使用连续噪声序列和GOOD_TCXO_V1，不重置信号相位。

名义时钟使用 `GOOD_TCXO_V1`：线性频漂 `0.020 Hz/s`、加加速度
`0.000020 Hz/s^2`、慢周期扰动 `0.50 Hz / 120 s`。

## 预期结果

不出现高频状态振荡；每次转换均有可解释的门限穿越和确认延迟。

统一精度门限为：载波有效观测率不低于 `95%`，多普勒 RMS 不超过
`5 Hz`、P95 不超过 `10 Hz`，动态真实码相位 P95 不超过 `0.20 chip`。
