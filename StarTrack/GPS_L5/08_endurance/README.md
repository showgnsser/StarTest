# GPS L5 - 一小时耐久测试

固定案例 ID：`ST-GPSL5-08-ENDURANCE`

## 现实场景

验证灵敏度边界下的长期数值稳定性、同步保持和误差漂移。

## 输入

- 信号：`GPS L5`
- 数据源：StarGen 实时二进制管道
- 单次时长：`3600 s`
- 固定种子：`20260716, 20260717, 20260718`
- 输入过程：分别在牵引、无辅助持续和已知位辅助三项边界运行3600秒。

## 真值

全程GOOD_TCXO_V1，实时二进制管道，不保存原始样点文件。

名义时钟使用 `GOOD_TCXO_V1`：线性频漂 `0.020 Hz/s`、加加速度
`0.000020 Hz/s^2`、慢周期扰动 `0.50 Hz / 120 s`。

## 预期结果

无长期失锁、异常周跳或持续漂移，观测有效率满足统一门限。

统一精度门限为：载波有效观测率不低于 `95%`，多普勒 RMS 不超过
`5 Hz`、P95 不超过 `10 Hz`，动态真实码相位 P95 不超过 `0.20 chip`。
