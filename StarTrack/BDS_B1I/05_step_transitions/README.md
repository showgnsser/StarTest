# BDS B1I - 强弱阶跃切换

固定案例 ID：`ST-BDSB1I-05-STEP_TRANSITIONS`

## 现实场景

验证强、中、弱、极弱和恢复过程中的状态响应及瞬态稳定性。

## 输入

- 信号：`BDS B1I`
- 数据源：StarGen 实时二进制管道
- 单次时长：`900 s`
- 固定种子：`20260716, 20260717, 20260718`
- 输入过程：强信号到中信号、弱信号、极弱信号，再恢复到强信号。

## 真值

C/N0按固定时刻阶跃，载波、码、数据位和噪声序列连续。

名义时钟使用 `GOOD_TCXO_V1`：线性频漂 `0.020 Hz/s`、加加速度
`0.000020 Hz/s^2`、慢周期扰动 `0.50 Hz / 120 s`。

## 预期结果

状态只在门限确认后切换，切换期间环路不发散且恢复后回到稳态。

统一精度门限为：载波有效观测率不低于 `95%`，多普勒 RMS 不超过
`5 Hz`、P95 不超过 `10 Hz`，动态真实码相位 P95 不超过 `0.20 chip`。
