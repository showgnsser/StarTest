# GPS L1CA - 缓慢遮挡与恢复

固定案例 ID：`ST-GPSL1CA-06-SLOW_FADE_RECOVERY`

## 现实场景

模拟建筑物或植被遮挡造成的缓慢衰减、驻留和恢复。

## 输入

- 信号：`GPS L1CA`
- 数据源：StarGen 实时二进制管道
- 单次时长：`900 s`
- 固定种子：`20260716, 20260717, 20260718`
- 输入过程：40 dB-Hz线性下降到目标极弱强度，保持后再线性恢复。

## 真值

信号幅度连续变化，噪声密度、载波和码相位连续。

名义时钟使用 `GOOD_TCXO_V1`：线性频漂 `0.020 Hz/s`、加加速度
`0.000020 Hz/s^2`、慢周期扰动 `0.50 Hz / 120 s`。

## 预期结果

无漏切换、无反复振荡；恢复延迟符合估计窗口和确认时间。

统一精度门限为：载波有效观测率不低于 `95%`，多普勒 RMS 不超过
`5 Hz`、P95 不超过 `10 Hz`，动态真实码相位 P95 不超过 `0.20 chip`。
