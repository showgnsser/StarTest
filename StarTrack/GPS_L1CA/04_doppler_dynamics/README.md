# GPS L1CA - 多普勒动态能力

固定案例 ID：`ST-GPSL1CA-04-DOPPLER_DYNAMICS`

## 现实场景

确定不同灵敏度下能够稳定承受的最大瞬时频率变化率。

## 输入

- 信号：`GPS L1CA`
- 数据源：StarGen 实时二进制管道
- 单次时长：`900 s`
- 固定种子：`20260716, 20260717, 20260718`
- 输入过程：扫描0、0.1、0.25、0.5、0.75和1 Hz/s，使用有界正弦动态而非无限斜升。

## 真值

GOOD_TCXO_V1基础扰动叠加指定最大频率变化率。

名义时钟使用 `GOOD_TCXO_V1`：线性频漂 `0.020 Hz/s`、加加速度
`0.000020 Hz/s^2`、慢周期扰动 `0.50 Hz / 120 s`。

## 预期结果

逐灵敏度给出最大通过频漂，失败边界保留为回归基线。

统一精度门限为：载波有效观测率不低于 `95%`，多普勒 RMS 不超过
`5 Hz`、P95 不超过 `10 Hz`，动态真实码相位 P95 不超过 `0.20 chip`。
