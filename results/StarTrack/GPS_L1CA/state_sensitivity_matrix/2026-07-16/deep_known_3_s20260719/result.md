# GPS L1CA - deep_known_3

## 测试目的

验证 Deep 状态在给定信号强度和连续多普勒动态下的稳态跟踪能力。

## 输入

- 时长：300 s
- 随机种子：20260719
- C/N0 模式：Step
- C/N0 时间表：0s:3.0dB-Hz
- 多普勒动态：线性 0.020 Hz/s，叠加 0.50 Hz、120 s 周期正弦扰动
- 捕获移交频差：0.100 Hz
- 捕获移交码相位误差：0.050 chip

## 预期结果

不重新捕获，不出现状态切换后持续发散；尾段频差 RMS 不超过 5 Hz，频差 P95 不超过 10 Hz，码相位误差 P95 不超过 0.2 chip。

## 实际结果

- 频差 RMS：1.785 Hz
- 频差 P95：3.201 Hz
- 码相位误差 P95：0.1522 chip
- C/N0 偏差：-0.207 dB
- C/N0 RMSE：3.589 dB
- FLL 锁定比例：100.0%
- DLL 锁定比例：100.0%
- 返回捕获次数：0
- 状态切换后发散次数：0
- 跟踪判定：通过
- C/N0 判定：未通过
- 状态判定：通过

## 结论

跟踪与状态切换通过，C/N0瞬态精度项未通过。
